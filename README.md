This is an interactive map experiment made for the third floor gardens of Clarion Hotel.
It features:
* A Full list of all the plants in the garden
* Map markers placed on each of the containers used in the garden, complete with the species of plants grown within.
* Searching for plants to see which containers have them.
* Separate lists for all types of container (Square, Rectangular, Circular and Triangular)
* Multi-lingual support ongoing.
* Planned feature: Individual pages for each plant, including a gallery of dishes that it is used in at the hotel.
* Planned feature: Up-close pictures of each container if you click their pin.
* Planned feature: Yearly change log, let the visitors see how the garden has grown year by year, keeping the old maps with their container placements, see which plants were grown each year and how the focus has changed.
